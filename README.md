# Project Title

E-commerce web site frontend UI.

## Description

This repository consists of frontend UI of an E-commerce website built using MERN Stack.

## Getting Started

### Dependencies

    @emotion/react,
    @emotion/styled,
    @mui/icons-material,
    @mui/material,
    @reduxjs/toolkit,
    axios,
    react,
    react-dom,
    react-redux,
    react-router-dom,
    react-stripe-checkout,
    redux-persist,
    styled-components

### Installing

npm install

### Executing program

node ./index.js
OR
nodemon ./index.js
